# 📜 古籍版本比对系统

一个**完全免费**、**无需安装**、**浏览器直接使用**的古籍研究工具。

## ✨ 功能特点

- 📤 支持上传古籍图片（JPG/PNG）和手动输入文本
- 🔍 内置免费OCR文字识别（支持简体中文）
- ✏️ 人工核查修改OCR结果
- 📊 多版本文本比对（逐句对照/差异高亮/并排对照）
- 📥 导出比对报告
- 🔌 支持自定义OCR API（百度/阿里云等）

## 🚀 使用方法

1. 访问：https://你的用户名.github.io/guji-compare/
2. 上传古籍图片或手动输入文本
3. 等待OCR自动识别
4. 人工核对修改识别结果
5. 选择多个版本进行比对

## 📖 注意事项

- 所有数据在浏览器本地处理，不会上传到服务器
- PDF文件建议先转换为图片再上传
- 古籍繁体字识别效果可能有限，建议人工核对

## 🔧 技术说明

- 前端：HTML5 + CSS3 + JavaScript
- OCR引擎：Tesseract.js（纯前端，免费）
- 部署：GitHub Pages
