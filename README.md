# 📜 古籍多版本比对系统

一个简单易用的在线古籍文本比对工具，专为古籍研究者、学生和爱好者设计。

## 🌐 在线访问
https://你的用户名.github.io/guji-version-comparison

## ✨ 功能特点
- 📷 **多格式上传**：支持图片（JPG/PNG）和PDF文件，每个文件最大200MB
- 🔢 **多版本比对**：最多支持6个版本同时比对
- 🤖 **智能OCR**：自动识别古籍文字，支持人工逐页核查修改
- 🔌 **API扩展**：可自定义使用其他OCR服务（百度、腾讯等）
- 📊 **直观比对**：高亮显示版本差异，提供统计信息
- 📥 **结果导出**：一键导出比对结果为文本文件
- 🆓 **完全免费**：无需注册，立即使用

## 🚀 快速开始
1. **访问网站**：打开上面的链接
2. **选择版本数**：在左侧选择要比对的版本数量（2-6个）
3. **上传文件**：为每个版本上传图片/PDF或直接输入文本
4. **设置OCR**：选择OCR识别方式（默认免费或自定义API）
5. **开始比对**：点击"开始OCR识别与比对"按钮
6. **查看结果**：系统显示比对结果，可导出或清空重来

## 📁 文件说明

- **纯前端实现**：无需服务器，保护隐私
- **响应式设计**：适配电脑、平板、手机
- **离线可用**：下载后可在本地使用
- **开源共享**：代码完全开放，可自由修改

## 📖 使用建议
1. **图片质量**：上传清晰的图片，识别效果更好
2. **文件大小**：单个文件不超过200MB
3. **人工核查**：建议启用"人工逐页核查"确保准确性
4. **版本命名**：为每个版本添加备注，便于区分
