# 免费古籍文本比对系统

一个简单易用的在线古籍文本比对工具，专为古籍研究者、学生和爱好者设计。

## 功能特点

- 📷 **图片文字识别**：上传古籍图片，自动提取文字
- 🔍 **智能文本比对**：高亮显示不同版本的差异
- 📊 **统计信息**：显示新增、删除字数及相似度
- 📥 **结果导出**：一键导出比对结果
- 🆓 **完全免费**：无需注册，立即使用

## 使用方法

1. **上传古籍图片**（支持JPG、PNG格式）
2. 或**直接输入文本**到文本框
3. 点击"开始比对"按钮
4. 查看高亮显示的差异结果
5. 可导出结果或清空重来

## 技术栈

- 前端：HTML5 + CSS3 + JavaScript
- OCR引擎：Tesseract.js（本地运行）
- 比对算法：Google diff-match-patch
- 部署：GitHub Pages

## 本地运行

直接双击 `index.html` 或在浏览器中打开即可。

## 在线访问

通过GitHub Pages部署后访问：https://你的用户名.github.io/guji-comparison

## 注意事项

1. 图片识别精度依赖图片清晰度
2. 首次使用OCR需要加载模型（约10MB）
3. 建议使用现代浏览器（Chrome、Edge、Firefox）
